# Credit Card Fraud - Dataset

This directory contains information about the dataset used in the **Credit Card Fraud Detection** project.

## Overview

- The original dataset is relatively large (~160 MB) and cannot be committed directly to this repository due to GitHub’s file size restrictions.
- It is a widely‑used dataset for fraud‑detection modelling, typically consisting of credit card transactions labeled as fraudulent or legitimate.

## Accessing the Data

To inspect or work with the dataset:

1. Open the `dataset url` file located in this directory.  
   It contains a direct download link (via GitHub Releases).
2. Download the CSV file.
3. Place it in the project root (or update the notebook path) before running any analysis notebooks.

> **Note:** Please review the hosting site’s terms of use before downloading.

## Usage

Once downloaded, the dataset can be explored and modelled via the `Credit_Card_Fraud.ipynb` notebook in the parent folder.  
The notebook includes data preprocessing, exploratory analysis, and classification examples.
